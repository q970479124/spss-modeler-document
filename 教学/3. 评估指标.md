# 一、方差与残差 #  
方差：解释原始数据的波动情况，它描述的是数据自身的波动性。  
> 公式：(真实值 − 平均值)² 的平均  
残差：解释预测数据和原始数据的偏离情况，它描述的是模型没解释掉的误差。  
> 公式：(真实值 − 预测值)  

##  ##

为什么方差要和残差做对比  
- 残差小说明模型准  
- 方差大说明数据分布广  
一个好的模型，就是让残差尽量比方差小很多，也就是：  
> 模型能解释掉大部分数据的波动。  
> 当残差远小于方差时，说明预测值的波动范围在原始数据的波动范围以内  
<br>

# 二、 什么是R² #  
R²就是1减去`残差平方和`和`总平方和`之间的比较  
**`残差平方和`和`总平方和`之间的比较说明模型不能解释的比例是多少**  
**用1减去得到的结果是能被模型解释的有多少**  
**越接近1说明模型能解释的越多**  
> 残差 = 真实值 - 预测值  
> 偏差 = 真实值 - 平均值  
> R² = 残差²和 比 偏差²和  
<br>

# 三、 MSE 和 RMSE #  
MSE 就是残差平方和的平均值，用来放大异常值  
> 残差 = 真实值 - 预测值  
> MSE = 残差²和的平均值  

##  ##

RMSE 就是残差平方和的平均值的开根号，用来放大异常值，单位恢复成原数据单位，更易理解  
> RMSE = MSE开根号  
<br>

# 四、 MAE 和 MAPE #
MAE 是残差的绝对值的平均值，不会放大异常值，只用来看平均差了多少  
> 残差 = |真实值 - 预测值|  
> MAE = 残差和的平均值  

##  ##

MAPE 是MAE的比例形式，以百分比的形式展示数据  
> 残差 = |真实值 - 预测值|  
> MAPE = 残差比真实值 * 100%  

<br>

# 五、 MAE 和 RMSE的对比  #

| 情况             | 模型特征          | 建议            |
| -------------- | ------------- | ------------- |
| RMSE ≈ MAE     | 误差分布均匀，模型稳定   | 表现好         |
| RMSE > MAE 一点点 | 有少量大误差        | 检查异常值      |
| RMSE ≫ MAE     | 存在明显大误差或模型不稳定 | 需改进模型或数据清洗 |
<br>

# 六、 偏差和方差的对比(评估模型效果) #

| 模型类型              | 偏差 | 方差 | 表现           | 问题特征        |
| ----------------- | -- | -- | ------------ | ----------- |
| 欠拟合（underfitting） | 高  | 低  | 无论训练集和测试集还是验证集的效果都很差 | 模型无法学习到数据中的规律   |
| 过拟合（overfitting）  | 低  | 高  | 训练集要远好于测试集和验证集  | 过拟合就是模型学习了`数据中的噪声`，而不是`真正的规律`  这部分噪声，只符合训练集，测试集和验证集不能拟合这部分规律，因此过拟合。 |
| 理想模型              | 低  | 低  | 训练与测试都好      | 学到规律且泛化良好   |

<br>

> 鲁棒性  
> 本质：模型的适应性，模型能够适应数据的多样性。
<br>

# 七、 熵 和 信息增益 #
熵： 案件的混乱程度（嫌疑太多，不确定谁是凶手）。   
- 不确定性度量  
信息增益： 一条线索能减少多少混乱。  
- 衡量特征“减少不确定性”的能力  
所以，一条“好线索”（高信息增益）会让我们更快接近真相。  

> 信息增益 = 分裂前的熵 − 分裂后的熵  
> 信息增益大 → 分裂让系统更有序 → 说明该特征能很好地区分数据。
