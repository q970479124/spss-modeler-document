## 如何评估模型 ##
### 评估模型效果 ###
1. 过拟合  
典型：训练集要远好于测试集和验证集  
本质：过拟合就是模型学习了`数据中的噪声`，而不是`真正的规律`  
这部分噪声，只符合训练集，测试集和验证集不能拟合这部分规律，因此过拟合。  

2. 欠拟合  
典型：无论训练集和测试集还是验证集的效果都很差  
本质：模型无法学习到数据中的规律  

3. 鲁棒性  
本质：模型的适应性，模型能够适应数据的多样性。

### 模型评估指标 ###
1. MAE
残差 = |真实值 - 预测值|  
MAE = 残差和的平均值  

2. MSE
残差 = 真实值 - 预测值  
MSE = 残差²和的平均值  

3. MAPE
残差 = |真实值 - 预测值|  
MAPE = 残差比真实值 * 100%

4. RMSE  
RMSE = MSE开根号

5. R²  
残差 = 真实值 - 预测值  
偏差 = 真实值 - 平均值  
R² = 残差²和 比 偏差²和
